# vue3_ts_three

本项目是使用vue3 + typescript + three.js 搭建的

项目结构
```
  ├─src
  │  ├─commponents
  │  │  ├─helpers three.js 辅助类
  |  │  └─models 模型加载类
  ├─app.ts 启动类
```
## 安装依赖

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

## 开发说明
App.ts为程序入口，以实例化后的app拆封功能并为各个组件提供服务
